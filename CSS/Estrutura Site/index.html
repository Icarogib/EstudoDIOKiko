Estudos HTML e CSS pela DIO 2022

Principais pontos:

  • Introdução:

    • Neste módulo, estamos nos familiarizando com dois sistemas web, muito importante para qualquer programador front-end. Vendo o básico, temos uma noção de como é formada a estrutura de um site em HTML pelas aulas ministradas pelo Professor Lucas Vilaboim.

  • HTML:

    • Com as aulas de HTML, sabemos que essa linguagem é responsável por ditar como o site é apresentado ao usuário, e em como é organizado. Podemos colocar links, textos, videos, fotos como é o exemplo atual, neste site feito com HTML e alterado pelo CSS (alteração nas cores e tamanhos de fonte.) Bases do HTML são variadas linhas como header, paragrafos, listas, imagens e outros (Muitos encontrados no material de apoio no GitHub no final da página).

  • CSS:

    • CSS é responsável por estilizar nossa página em HTML, fazendo alteração na "caixa", nas letras (como mudança de cor da fonte e tamanho). Utilizando background nas classes e body, mudamos a cor de fundo, para mudar das letras (headers), utilizamos color. É possível também a introdução de bordas e espaçamento dessas bordas com, respectivamente, border e margin, dentre outras funções.
